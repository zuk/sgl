h1. sgl

h1. &#x2192; simple generic library

h2. What

'sgl' is a simgple generic library to create a program with graphics and sound.

h2. Requirement

h3. Windows
* Ruby 1.8.5 or later
* Ruby/OpenGL
* Ruby/Math3D
* Ruby/SDL

You can see a document to install these libraries here.
http://eto.com/wiki/RubySDLonWindows.html

h3. Mac OS X
* Mac OS X 10.3 or later
* Ruby 1.8.5 or later
* X Window system

h2. Installing

<pre syntax="ruby">sudo gem install sgl</pre>

h2. The basics

Save this progarm as <tt>test1.rb</tt>.
<pre syntax="ruby">
require "sgl"
window 100, 100
line 0, 0, 100, 100
wait
</pre>

Run the program with <tt>ruby test1.rb</tt>.

<pre syntax="ruby">
require "sgl"

def setup
  window 100, 100
end

def display
  line 0, 0, 100, 100
end

mainloop
</pre>
