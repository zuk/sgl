sgl-cocoa → sgl with Ruby/Cocoa

Get Version

0.3.1

What

‘sgl-cocoa’ is a cocoa version of sgl. You can use sgl functions with Ruby/Cocoa.

Requirement

You can use this only on Mac OS X.

Mac OS X

  • Mac OS X 10.3 or later
  • ruby 1.8.5 or later
  • RubyCocoa 0.4.1 or later

The basics

Save this progarm as test1.rb. 
require "sgl/cocoa" 

def setup
  window 100, 100
end

def display
  line 0, 0, 100, 100
end

mainloop

Run the program with ruby test1.rb.
